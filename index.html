OXELATE SECURITY BOT - PRIVACY POLICY

Last Updated: June 2026

Your privacy is highly valued. This Privacy Policy outlines what data Oxelate processes, stores, and your rights regarding that data.

1. DATA WE COLLECT

To provide high-tier security, Anti-Nuke, and Auto-Mod functionalities, the Bot processes and stores the following data:

  • Guild/Server Identifiers: Server IDs, Channel IDs, Role IDs, and custom configuration settings.
  • User Identifiers: Discord User IDs of server members, administrators, and whitelisted users.
  • Action Logs: Temporary moderation logs, message counts, and action histories to detect and prevent unauthorized server modifications (nuking/raiding).

2. HOW WE USE THE DATA

The collected data is strictly used for the execution of the Bot's core security features:

  • Enforcing Anti-Nuke limits and identifying unauthorized actions.
  • Executing Auto-Mod protocols and maintaining custom server configurations.
  • Verifying whitelisted users and handling support requests.

3. DATA RETENTION & SECURITY

  • All configurations and logs are stored securely using encrypted cloud databases (MongoDB Atlas).
  • Data is retained only as long as necessary to serve the functionality of the Bot or until the Bot is removed from the server.

4. DATA DELETION REQUESTS

If you wish to wipe all data related to your server or user ID from our database, you can:

  • Remove (kick) the Bot from your server, which will automatically mark the server configuration data for deletion.
  • Open a support ticket in the official Oxelate Headquarter server to request an immediate manual data wipe from our logs.

5. THIRD-PARTY DISCLOSURE

We do not sell, trade, share, or transfer your data to any third-party individuals or external services. All data stays strictly within the secure Oxelate ecosystem.
